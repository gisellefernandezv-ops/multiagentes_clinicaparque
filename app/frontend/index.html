📖 Documentación

Cómo usar InvoiceFlow

3 formas de procesar facturas

  1. Automática (watcher): Copiá archivos en platform/data/inbox/
  2. Manual por chat: Escribí en el chat interno "procesá todo el inbox"
  3. Manual por UI: Andá a la pestaña Inbox y procesá manualmente

Formatos aceptados

{
  "invoice_id": "INV-2024-001",
  "supplier_id": "SUP001",
  "supplier_name": "TechCorp SA",
  "amount": 50000,
  "currency": "ARS",
  "invoice_date": "2025-06-01"
}

Endpoints API

  • GET /health — Estado de microservicios
  • GET /inbox — Lista facturas en inbox
  • POST /inbox/upload — Sube factura
  • POST /inbox/process/<filename> — Procesa una
  • GET /invoices — Historial
  • GET /dashboard — Stats
  • POST /chat — Chat-driven
